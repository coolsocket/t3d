# AGENTS.md

This repo follows the [AGENTS.md convention](https://agents.md). Canonical
instructions for AI coding agents working in this repo are in
[`CLAUDE.md`](./CLAUDE.md).

If you're a non-Claude agent (Cursor, Aider, Codex, Cline, etc.) editing
this repo, the rules in `CLAUDE.md` still apply — they aren't Claude-specific.

If you're a user trying t3d in YOUR project, you don't need this file. Run
`/t3d-init` in your project and follow the `CLAUDE.md` that lands there.
